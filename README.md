# Booking System API

API REST desarrollada con NestJS para la gestión de un sistema de reservas de propiedades tipo Airbnb.

Incluye autenticación segura con JWT, control de acceso basado en roles, gestión de reservas, propiedades y reviews, así como lógica avanzada de negocio como prevención de overbooking y cancelación automática de reservas.

## 🚀 Tecnologías
- NestJS
- Prisma ORM
- PostgreSQL
- JWT Authentication
- Event Emitter
- Cron Jobs

## 🔥 Características
- Autenticación y autorización con roles (ADMIN, HOST, GUEST)
- Sistema de reservas con validación de fechas
- Prevención de overbooking
- Confirmación y cancelación de reservas
- Auto-cancelación de reservas pendientes
- Sistema de reviews con rating promedio
- Eventos y notificaciones por email
- Arquitectura modular y escalable
